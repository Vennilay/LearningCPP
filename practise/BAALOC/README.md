# Инструкция по работе с практическими заданиями

## Твоя папка и права доступа

Ты находишься в папке `Practise/[твой_ник]/` — это **твоя личная папка** для практических заданий.

### Что ты можешь делать:
- ✅ Создавать, редактировать и удалять **любые файлы** внутри `Practise/[твой_ник]/`
- ✅ Создавать подпапки `PR_1/`, `PR_2/`, `PR_3/` и так далее для разных практик
- ✅ Открывать Pull Request'ы с изменениями в своей папке
- ✅ Одобрять (approve) свои собственные PR для слияния в `main`

### Что ты НЕ можешь делать:
- ❌ Редактировать файлы в папках других студентов (`Practise/Vennilay/`, `Practise/BAALOC/` и т.д.)
- ❌ Пушить напрямую в ветку `main` — все изменения идут через Pull Request
- ❌ Смержить PR, который затрагивает чужие папки, без одобрения их владельцев
