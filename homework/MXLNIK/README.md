# Инструкция по работе с репозиторием

## Твоя папка и права доступа

Ты находишься в папке `homework/[твой_ник]/` — это **твоя личная папка** для выполнения домашних заданий.

### Что ты можешь делать:
- ✅ Создавать, редактировать и удалять **любые файлы** внутри `homework/[твой_ник]/`
- ✅ Создавать подпапки `HM_1/`, `HM_2/`, `HM_3/` и так далее для разных заданий
- ✅ Открывать Pull Request'ы с изменениями в своей папке
- ✅ Одобрять (approve) свои собственные PR для слияния в `main`

### Что ты НЕ можешь делать:
- ❌ Редактировать файлы в папках других студентов (`homework/Vennilay/`, `homework/BAALOC/` и т.д.)
- ❌ Пушить напрямую в ветку `main` — все изменения идут через Pull Request
- ❌ Смержить PR, который затрагивает чужие папки, без одобрения их владельцев
